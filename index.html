Is Machining Necessary?

Before diving into the nitty-gritty of CNC machining a part, it is important to ask yourself if it is necessary. Often a part or assembly can be designed in such a way that in-house machining is greatly reduced or removed entirely from the process. Some common motivaitons for machining parts include:

  • Precision features (flatness on seal faces, precision holes, engineered fits)
  • Material strength or stiffness (aerospace, automotive)
  • Extreme temperature applications (rocket parts, superconducting magnet parts)

After deeming machining necessary, it is time to decide if Turning or Milling better suits the manufacturing of the part. Milling is well suited to parts that can be described in the cartesian coordinate system such as curved shapes in a plane or 3d contours. Turning is suited more to parts that can be described in the cylindrical coordinate system such as shafts or parts with a cylindrical surface. When designing parts it is best practice to have the manufacturing process in mind. Even when 3d printing it is helpful to visualize how each of your parts might be manufactured in a mill or lathe.

4.1: Switch to the Manufacturing Tab

First it is necessary to open the part as a .STEP file in Fusion and save it. Then, clicking the manufacturing tab will bring the user to the CAM user interface where the rest of the tutorial will take place.

4.2: Defining Stock and the WCS (Work Coordinate System)

Now it is time to define the material (stock) that you are using in real life virtually so that the the machine can be programmed to remove it correctly without any crashes. To do this, first measure the stock in real life, then click setups, new setup.

Now it is time to define the coordinate system. In most cases it is convinent for the coordinate system to be defined off of a stock corner as this is easy to locate with the machine in real life.

I have chosen the following coordinate system based on how I envision i will hold the part in the vice on the mill. learning this comes with practice but it can be picked up quickly.

Now, the stock box must be defined correctly with respect to the measurements of the stock in real life. There are ways around this, but it is simple to just input the actual size to avoid crashes. (also remember to square up the stock before machining!)

After defining the stock size and the WCS (work coordinate system), it is time to move onto the fun part - toolpathing!

4.4: Toolpathing

Now for the fun part, programming the operation to best subtract away material from the stock as it is defined, leaving your part. For this example part, it is clear that the operation type needed for the large pocket in the middle of the part is a 2d pocket. Common operation types are 2 and 3D pocketing, boring, 2 and 3D contouring, and facing. The setup for all of the different types is simple and clear to click through with Fusion.

Now it is time to select a tool for the toolpath from the tool libary. Because this pocket has a flat bottom with no chamfer on its edge, a flat end mill was chosen. The desired material to make this part is aluminum, so a two flute carbide end mill was chosen. Click on the select button next to tool in the above image. After navigating to the El Dorado tool library, select a suitable tool After navigating to the El Dorado tool library, select a suitable tool.

In the absense of a coolant system no coolant can be used, so no need to uncheck this box as the coolant will never be fired off. Coolant is available on many other machines and allows for faster cycle times and often better surface finishes.

Next, move to the geometry tab and select the feature of interest. In this case it is the lower face of the pocket. There are different ways to specify a feature and fusion is pretty forgiving.

Now that the geometry has been selected, it is time to evaluate other parameters of the toolpath. Two important adjustments are the number of passes and if you want to have a "finishing" pass. The default setting for a pocket operation is for the toolpath to be completed at one depth. While this is theoretically possible sometimes, it is not a best practice as the cutting force is high and machine and tool wear are increased. The main issue with doing this on this machine is that the spindle motor may stall and ruin the part or you may break an end mill with too high of a cutting force.

This leaves us at the multiple depths setting to solve this issue. Typically a safe cut depth is 1-2mm for this machine and/or a new user. This can be less for harder materials such as steel and greater for softer materials such as plastics. Sometimes this value is auto-populated based on tool data.

Finally you can hit ok after realizing that a finishing pass is not needed for your part and then view and animate the simulated toolpath.

If the part needs more than one operation to finish (like this one), then adding another operation is simple.

Select the type of operation that is needed from the navbar menu and then go through the dialog again. The feature of interest that was not machined with the inital pocket operation is the small "open pocket" on the bottom edge. This feature can be machined with a 2d countour operation.

The steps are similar to the pocket operation: Select operation type, choose tool from tool library, choose geometry to machine, then modify features such as stepdown amount and if there is a finishing pass.
